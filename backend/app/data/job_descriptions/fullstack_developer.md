# Full-Stack Developer — ABC Company (Sample JD)

We're looking for a Full-Stack Developer to help build and maintain our
customer-facing web products end to end.

## Responsibilities
- Build features across the stack: React frontend, Node/Python backend.
- Design clean REST APIs and data models.
- Write maintainable, tested code and participate in code review.
- Debug production issues and improve performance.

## Requirements
- Solid understanding of data structures & algorithms.
- Experience with a modern frontend framework (React preferred).
- Experience with backend frameworks (Express, FastAPI, Django, etc).
- Comfortable working with REST APIs and relational databases.

## Nice to have
- Experience with CI/CD and cloud deployment.
- Experience mentoring or leading small project teams.
